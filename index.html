Differnces Between Web 1.0, Web 2.0 and Web 3.0

  1. Interaction
    • Web 1.0: Mostly read-only without a lot of interaction between end user
    • Web 2.0: Wildly read-write and tend to interact much more with the end user
    • Web 3.0: Portable and personal which is used to describe many evolutions of web usage and interaction among several paths. In this, data isn’t owned but instead shared, where services show different views for the same web / the same data
  2. Usages
    • Web 1.0: Company focus
    • Web 2.0: Community Focus
    • Web 3.0: Individual Focus
  3. Advertising
    • Web 1.0: Banner advertising - advertisements on websites while surfing the internet is banned
    • Web 2.0: Interactive advertising - An interaction and collaboration with each other is allowed by in a social media dialogue as creator of user-generated content in a virtual community
    • Web 3.0: Behavioural advertising
